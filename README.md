 📚 Smart Study Assistant

 Overview

Smart Study Assistant is a Flask-based web application designed to help students revise study materials more effectively.

The application allows users to upload PDF notes, automatically generates revision questions from the document content, collects self-assessment responses, calculates a study score, and identifies areas that require further revision.

 Features

* Upload PDF study materials
* Automatic text extraction using PyPDF
* Revision question generation from PDF content
* Self-assessment using confidence levels
* Study score calculation
* Revision recommendations
* Dark Mode and Light Mode support

 Technologies Used

* Python
* Flask
* PyPDF
* HTML
* CSS

 How It Works


1. Upload a PDF file.
2. The application extracts text from the document.
3. Important content is converted into revision questions.
4. The user evaluates their understanding of each question.
5. A study score is calculated.
6. Weak areas are highlighted for further revision.

 Project Structure

Smart Study Assistant helps students quickly convert study notes into interactive revision quizzes and monitor their learning progress.

 Future Enhancements

* AI-generated questions
* Multiple quiz formats
* Downloadable study reports
* Personalized revision plans
